Features Overview

A concise summary of the app and its purpose

This is a full-stack blog application built using Python, Flask, SQLAlchemy, and Bootstrap. It allows users to create and manage blog posts and comments in a secure and user-friendly environment. The interface is responsive and works seamlessly across different devices.

Tech Stack

  • Backend: Python, Flask, SQLAlchemy
  • Frontend: HTML, CSS, Bootstrap (customised from a template), JavaScript (AJAX for dynamic comment handling)
  • Authentication: Flask-Login for user sessions and access control
  • Database: SQLite (using SQLAlchemy as the ORM)

User Roles & Permissions

  • Admin User: Full control over all content (can edit/delete any user's posts and comments)
  • Regular Users: Can manage only their own posts and comments
  • Guests: Can view blog posts and comments but cannot create or modify content

Blog Post Features

  • Create blog posts with a title, subtitle, body, and background image
  • View blog posts created by any user
  • Edit or delete your own posts

Commenting System

  • AJAX-powered comment creation, editing, and deletion for an efficient commenting experience without full page reloads
  • Comments are displayed beneath each post

Authentication & Access Control

  • Users must register and log in to create, edit, or delete posts and comments
  • Authorisation logic prevents users from modifying content they donâ€™t own
  • Login errors include wrong password and unregistered email handling
  • Registration redirects to login if the email is already in use

Responsive Design

  • Optimised for desktop, tablet, and mobile viewing
  • Styled with a customised Bootstrap template for a clean and intuitive UI
